# Варіанти дизайну Header

## Варіант 1: Мінімалістичний з балансом
- Простий, чистий дизайн
- Баланс користувача поруч з профілем
- Менші елементи, більше простору

## Варіант 2: Градієнтний фон
- Яскравий градієнтний фон header
- Більш виразні кольори
- Сучасний, привабливий вигляд

## Варіант 3: З акцентом на профіль
- Великий профіль користувача по центру
- Баланс та додаткова інформація
- Більш особистий підхід

## Варіант 4: Темна тема з неоновим ефектом
- Темний фон з неоновим підсвічуванням
- Кольорові акценти на елементах
- Футуристичний вигляд

## Варіант 5: З додатковими кнопками
- Кнопки навігації (налаштування, повідомлення)
- Більше функціональності
- Більш повноцінний header

## Варіант 6: Компактний з іконками
- Менші елементи
- Більше інформації в меншому просторі
- Оптимізований для мобільних
